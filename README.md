# 🖥️ Stats98 — Мировое Бюро Статистики

> An interactive retro data portal that looks and feels like a forgotten website from 1998. Built with pure HTML/CSS/JS — no frameworks, no dependencies.

## ✨ Features

🪟 **Draggable Windows** — Full Windows 98 UI with title bars, minimize, close, and drag
📊 **Interactive Charts** — Bar charts, pie charts, and mini graphs — all pure CSS/JS
🔢 **Live Population Counter** — Odometer-style counter ticking up in real-time
💣 **Minesweeper of Knowledge** — Click cells to discover facts, avoid the bombs!
🎭 **Nickname Generator** — Create your 90s internet name in Russian or English
🔊 **Windows 98 Sounds** — Startup, clicks, errors, explosions — all via Web Audio API
⚠️ **Fake Error Dialogs** — "IEXPLORE caused an error in module STATS98.DLL"
📺 **Dial-up Loading Screen** — Full modem connection simulation at 28.8 Kbps
🖱️ **Custom Cursor** — Pixel-perfect Windows 98 arrow cursor
📌 **Start Menu & Taskbar** — Fully functional with window switching
👁️ **CRT Effects** — Scanlines, vignette, retro color palette

## 🚀 Run Locally

Just open the file:

```bash
open index.html
```

Or use any static server:

```bash
npx serve .
```

## 🌐 Deploy

It's a single HTML file. Deploy anywhere:

- **Vercel** — drag & drop or `vercel --prod`
- **Netlify** — drag & drop the folder
- **GitHub Pages** — enable in repo settings

## 🛠️ Tech

- **Zero dependencies** — pure HTML + CSS + JavaScript
- **Web Audio API** — all sounds generated in code, no audio files
- **SVG cursors** — embedded as data URIs
- **Google Fonts** — Press Start 2P, VT323, IBM Plex Mono
